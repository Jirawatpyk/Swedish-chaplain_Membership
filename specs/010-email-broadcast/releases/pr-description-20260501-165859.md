# F7 US3 — Member quota dashboard + history + Q15 banner

## Summary

Ships F7 User Story 3 of the Email Broadcast (E-Blast) feature: paying-tier members can now see their annual e-blast entitlement, year-to-date used / reserved / remaining counts, the next reset date in tenant timezone, a paginated history of their own broadcasts, a per-broadcast detail page with delivery breakdown (delivered / bounced / soft-bounced / complained), and a GDPR Art. 7 acknowledgement banner.
