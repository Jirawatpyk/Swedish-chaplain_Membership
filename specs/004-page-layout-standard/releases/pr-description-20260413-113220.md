## Implementation

**New primitives** (`src/components/layout/`, `src/components/shell/`):
- `PageHeader` — title/subtitle/actions/badge, CSS logical properties for RTL readiness
- `ContentContainer` — admin 72rem / portal 64rem variants
- `BreadcrumbNav` + `BreadcrumbProvider` + `BreadcrumbPath` — depth≥3 rule, mobile truncation, i18n labels, percent-encoded URL round-trip
- 6 skeleton primitives — `SkeletonBlock`, `CardSkeleton`, `FormSkeleton`, `TableSkeleton`, `DetailSkeleton`, `PageSkeletonShell`

**Suspense strategy** — single route-level boundary:
- 11 `loading.tsx` files render real `<PageHeader>` + `<Card>` chrome with skeleton only on dynamic data. `<PageSkeletonShell>` is the sole `role="status" aria-live="polite"` live region per page (nested live regions avoided per ARIA best practice).
- Internal `<Suspense>` in `/admin/plans/page.tsx` + `/admin/users/page.tsx` **removed** — double-wrapping caused a visible two-pass shimmer on navigation.

**Design tokens** (`globals.css`): 30 new CSS variables covering spacing, max-widths, top bar, typography scale (h1–h4 + body + caption + `--line-height-th: 1.65`), form fields, data tables, cards, modals, plus `.skeleton-shimmer` utility respecting `prefers-reduced-motion`.

**shadcn/ui primitive updates** — `button.tsx` (h-8→h-9 + cursor-pointer + disabled), `input/textarea/select/label.tsx` (form-field tokens), `table.tsx` (sticky thead + focus-within mirrors hover + Thai line-clamp opt-in), `card.tsx` (padding/radius/shadow tokens + dark override), `dialog/alert-dialog/sheet.tsx` (modal tokens). See `docs/shadcn-customizations.md` for the full diff catalogue.

**Error boundaries** — `src/app/(staff)/admin/error.tsx` + `src/app/(member)/portal/error.tsx` protect against server-component failures (e.g. Neon timeout in `UsersDataSection` / `PlansList`) with reset() + digest-based log correlation.

**ESLint** — `no-restricted-syntax` rule blocks ad-hoc utility classes (`max-w-*`/`mx-auto`/`container`/`p-*`/`text-(xl|2xl|3xl|4xl)`) on page.tsx root elements (FR-003 enforcement).
