# UAT: จัดการแพ็กเกจสมาชิก (Membership Plans) — list / create / clone / edit / activate / delete / palette (F2)

> **คู่มือ flow นี้:** [../../user-guide/admin/membership-plans.md](../../user-guide/admin/membership-plans.md)
> **ที่มา:** `specs/002-membership-plans/spec.md` (US1–US6) + Success Criteria
> **รันบน:** preview deploy (ไม่ใช่ production) · บัญชี: admin, manager, member

## ก่อนเริ่ม (Preconditions รวม)
- [ ] มี seed แพ็กเกจ SweCham 2026 ครบ **9 ตัว** (corporate 6 + partnership 3) สถานะถูกต้อง
- [ ] การตั้งค่าใบกำกับภาษี (Settings → Invoicing) มีค่าสกุลเงิน THB / VAT 7% (เพื่อให้ list แสดงสกุลเงิน + ยอด VAT preview ได้)
- [ ] มีบัญชีทดสอบครบ 3 บทบาท: **admin**, **manager**, **member**
- [ ] ปีปลายทางสำหรับทดสอบ Clone (เช่น 2030) **ยังไม่มีแพ็กเกจ** (เพื่อทดสอบ clone สำเร็จ)

**วิธีกรอก:** แต่ละ TC ทำเครื่องหมาย ✅ ผ่าน / ❌ ไม่ผ่าน ในช่อง "ผล" + ใส่หลักฐาน (ชื่อ/ปีแพ็กเกจที่ได้ / ภาพหน้าจอ) ในช่อง "หมายเหตุ"
