# UAT: เข้าสู่ระบบ / สิทธิ์ผู้ใช้ (F1 — Auth & RBAC)

> **คู่มือ flow นี้:** [../../user-guide/admin/auth-rbac.md](../../user-guide/admin/auth-rbac.md)
> **ที่มา:** `specs/001-auth-rbac/spec.md` (US1–US7) + Success Criteria (SC-001…SC-022)
> **รันบน:** preview deploy (ไม่ใช่ production) · บัญชี: admin, manager, member

## ก่อนเริ่ม (Preconditions รวม)
- [ ] มีบัญชี **admin** ที่ active อย่างน้อย 1 บัญชี (รู้รหัสผ่าน) — สร้างจากสคริปต์ bootstrap
- [ ] มีบัญชี **manager** ที่ active 1 บัญชี (สำหรับเทสสิทธิ์อ่านอย่างเดียว)
- [ ] มีบัญชี **member** ที่ active 1 บัญชี + อีเมลทดสอบที่เปิดอ่านได้จริง (รับลิงก์เชิญ/รีเซ็ต)
- [ ] อีเมลขาออก (Resend) ใช้งานได้บน environment ที่ทดสอบ
- [ ] รู้ URL ทั้งสอง portal: `/admin/sign-in` (staff) และ `/portal/sign-in` (member)

**วิธีกรอก:** แต่ละ TC ทำเครื่องหมายในช่อง "ผล" (☐ ผ่าน หรือ ☐ ไม่ผ่าน) + ใส่หลักฐาน (ภาพหน้าจอ/ข้อความที่ขึ้น/เวลา) ในช่อง "หมายเหตุ"

## TC-AUTH-18 — เปิดบัญชีกลับ (Re-enable)
**อ้างอิง:** US4-AS3 (re-enable), FR-010 · **บทบาท:** admin → ผู้ถูกเปิด

| # | ขั้นตอน | ผลที่คาดหวัง |
|---|---|---|
