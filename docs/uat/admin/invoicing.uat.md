# UAT: ออกใบกำกับภาษี / ใบเสร็จ / ใบลดหนี้ / ยกเลิก (F4)

> **คู่มือ flow นี้:** [../../user-guide/admin/invoicing.md](../../user-guide/admin/invoicing.md)
> **ที่มา:** `specs/007-invoices-receipts/spec.md` (US1/US2/US5/US6) + Success Criteria
> **รันบน:** preview deploy (ไม่ใช่ production) · บัญชี: admin, manager, member

## ก่อนเริ่ม (Preconditions รวม)
- [ ] ตั้งค่า **Settings → Invoicing** ครบ (ชื่อนิติบุคคล TH/EN, เลขภาษี 13 หลัก, ที่อยู่ TH/EN, VAT 7%)
- [ ] มีบริษัทสมาชิกทดสอบ ≥ 1 ราย และมีแพ็กเกจ (plan) ปีปัจจุบัน active
- [ ] `FEATURE_F4_INVOICING` เปิด

**วิธีกรอก:** แต่ละ TC ทำเครื่องหมาย ✅ ผ่าน / ❌ ไม่ผ่าน ในช่อง "ผล" + ใส่หลักฐาน (เลขเอกสารที่ได้/ภาพหน้าจอ) ในช่อง "หมายเหตุ"

## TC-INV-01 — ออกใบกำกับภาษีเต็มปี + VAT 7%
**อ้างอิง:** US1-AS1, SC-002 · **บทบาท:** admin

| # | ขั้นตอน | ผลที่คาดหวัง |
|---|---|---|
| 1 | Invoices → New invoice → เลือก radio **Membership** (legend "What is this invoice for?") | ฟอร์มเปิด มีช่อง Member / Plan / วันที่ / line items |
| 2 | เลือกสมาชิก corporate, plan ปีนี้, issue date 2026-01-15 | ระบบเติมรายการค่าสมาชิก (+ ค่าแรกเข้า ถ้าสมาชิกใหม่) |
| 3 | ดูตัวอย่างยอดท้ายฟอร์ม | แสดง **VAT 7%** บน subtotal + ยอดรวมถูกต้อง |
| 4 | บันทึกร่าง → กด Issue… → พิมพ์คำว่า **"ISSUE"** ในกล่องยืนยัน → กด Issue | สถานะ **Issued**, ได้เลข `INV-2026-0000xx` ไม่ซ้ำในปี |
| 5 | ดาวน์โหลดผ่านเมนู "⋯" (More) → Download invoice PDF | PDF เป็น **ไทย** มีคำแปลอังกฤษทุกป้าย, ป้าย "ใบกำกับภาษี / Tax Invoice", มีชื่อ+เลขภาษีทั้งผู้ขาย/ผู้ซื้อ, รายการ, subtotal+VAT+total, เลขรัน, วันที่ (ค.ศ.+พ.ศ.) |

**ผล:** ☐ ผ่าน ☐ ไม่ผ่าน — **หมายเหตุ (เลขที่ได้/หลักฐาน):** ____________________
