# คู่มือสมาชิก: เริ่มต้นใช้งาน Member Portal (MEMBER)

> **ใครใช้:** **สมาชิก** ของหอการค้า (สิทธิ์ `member`) — บัญชีที่ผูกกับบริษัทสมาชิกหนึ่งราย
> **เมนู:** Dashboard (`/portal`) · Profile (`/portal/profile`) · Invoices (`/portal/invoices`) · Benefits (`/portal/benefits`) · Timeline (`/portal/timeline`) · Account (`/portal/account`)
> **UAT ของ flow นี้:** [../../uat/member/quick-start.uat.md](../../uat/member/quick-start.uat.md)

> 🌐 **หมายเหตุภาษา:** คู่มือนี้เขียน **คำอธิบายเป็นไทย** แต่อ้างชื่อปุ่ม/เมนูเป็น **อังกฤษ** ตรงตามที่แอปแสดงในโหมด English (ค่า default `en.json`) เพราะสมาชิกบางส่วนเป็นบริษัทสวีเดน/ต่างชาติ หากต้องการคู่มือฉบับ **อังกฤษ/สองภาษา** สำหรับสมาชิกต่างชาติ แจ้งได้ — สมาชิกเปลี่ยนภาษาแอปเองได้ที่ Account → Notification language (ดูงานที่ 6)

---

## ⚠️ ก่อนเริ่ม — สิ่งที่ต้องมี

1. **ได้รับอีเมลเชิญ (invitation email)** จากเจ้าหน้าที่หอการค้าแล้ว — บัญชีสมาชิกสร้างจาก "คำเชิญ" เท่านั้น ไม่มีการสมัครเอง
2. คำเชิญ **ยังไม่หมดอายุ และยังไม่เคยถูกใช้** (ลิงก์ใช้ครั้งเดียว) — ถ้าหมดอายุ/ถูกใช้แล้ว ให้แจ้งเจ้าหน้าที่ออกคำเชิญใหม่
3. บัญชีของคุณ **ถูกผูกกับบริษัทสมาชิก (member)** แล้ว — ถ้ายังไม่ผูก หน้าจอจะขึ้นข้อความ *"Your account is not linked to a member. Please contact your administrator."* ให้ติดต่อเจ้าหน้าที่
4. เข้าสู่ระบบสมาชิกที่ **`/portal/sign-in`** (คนละหน้ากับเจ้าหน้าที่ที่ `/admin/sign-in`)
5. บางเมนูจะแสดงเฉพาะเมื่อหอการค้าเปิดฟีเจอร์ไว้:
   - **Pay now** (จ่ายออนไลน์) — แสดงเมื่อหอการค้าเปิด online payment + บัตร/PromptPay (F5)
   - **E-Blasts / Benefits → Broadcasts** — แสดงเมื่อเปิด E-Blast benefit (F7)
   - **Data & privacy (export ข้อมูล)** — แสดงเมื่อเปิด F9 dashboard
   - **Directory listing** — แสดงเมื่อเปิด F9 dashboard

---

## ภาพรวม — สมาชิกทำอะไรได้บ้าง (จำให้แม่น)

```
อีเมลเชิญ ──Activate account──► เข้าสู่ระบบ /portal
                                   │
   ┌───────────────┬──────────────┼───────────────┬────────────────┐
   ▼               ▼              ▼               ▼                ▼
Dashboard       Profile        Invoices        Benefits         Account
(ภาพรวม)        (ดู+แก้บางช่อง)  (ดู/จ่าย/โหลด)   (โควตา/E-Blast)   (รหัสผ่าน/ภาษา/
                                   │                              export/ออกจากระบบ)
                              [Issued]──Pay now──►[Paid]──Download receipt
                                   │
                              Download invoice PDF
```

> 🔑 **กฎเหล็ก #1 — เห็นได้เฉพาะบริษัทของคุณ:** สมาชิกเห็นเฉพาะข้อมูลบริษัทตัวเอง (โปรไฟล์ ใบแจ้งหนี้ สิทธิประโยชน์ timeline) เท่านั้น — พยายามเปิด URL ของบริษัทอื่นจะได้ **404 / not found** เสมอ
> 🔑 **กฎเหล็ก #2 — แก้ได้แค่บางช่อง:** สมาชิกแก้ได้เฉพาะ ชื่อ-นามสกุล / เบอร์โทร / ภาษา / website / description เท่านั้น ส่วน plan, tax ID, สถานะ, ประเภทนิติบุคคล, อีเมล เป็นของเจ้าหน้าที่ (admin) แก้ผ่านพอร์ทัลไม่ได้

## งานที่ 1 — เปิดใช้งานบัญชีจากอีเมลเชิญ (ครั้งแรกครั้งเดียว)

**สถานการณ์:** เพิ่งได้อีเมล "Finish setting up your account" จากหอการค้า

1. เปิดอีเมลเชิญ → กดลิงก์ในอีเมล (พาไปหน้า `/invite/[token]`) — หัวข้อหน้า **"Finish setting up your account"**
2. ตรวจ **Email** (เติมมาให้แล้ว — เปลี่ยนไม่ได้) → กรอก **Your name** (ชื่อที่จะแสดง)
3. ตั้งรหัสผ่านในช่อง **Choose a password** — ระบบตรวจความแข็งแรงของรหัส (อย่างน้อย 12 ตัวอักษร และต้องไม่อยู่ในรายการรหัสที่เคยรั่วไหล)
4. กด **"Activate account"** → ระบบสร้างบัญชี + เข้าสู่ระบบให้ → เด้งไปหน้า **Dashboard** (`/portal`)

> ⚠️ ลิงก์เชิญ **ใช้ได้ครั้งเดียวและมีวันหมดอายุ** — ถ้าขึ้น *"This invitation has expired"* หรือ *"This invitation has already been used"* ให้แจ้งเจ้าหน้าที่ออกคำเชิญใหม่ (ปุ่ม **"Contact an administrator to request a new invitation"**)
> ℹ️ ครั้งถัดไปให้เข้าที่ **`/portal/sign-in`** ด้วย Email + Password ที่ตั้งไว้

---

## งานที่ 2 — ดูภาพรวมสมาชิก (Dashboard) + สถานะ/สิทธิประโยชน์

**สถานการณ์:** เข้าระบบมาดูว่าสมาชิกภาพเรายังดีอยู่ไหม ค้างจ่ายอะไรหรือเปล่า

1. เข้า **Dashboard** (`/portal`) — หัวข้อทักทาย **"Hi {ชื่อ}"** พร้อม badge **เลขสมาชิก** (เช่น `SCCM-0042`) และป้ายสถานะ **Active / Inactive / Archived**
2. ดู 3 การ์ดสรุปด้านบน:
   - **Membership** — สถานะสมาชิก (Active / Renew soon / Renewal overdue / ฯลฯ)
   - **Outstanding balance** — ยอดค้างชำระ + จำนวนใบที่ยังไม่จ่าย (ถ้าจ่ายครบจะขึ้น **"All paid"**)
   - **Benefits** — สิทธิประโยชน์ที่ใช้ไม่คุ้ม (under-used) หรือ **"All benefits on track"**
3. ด้านล่างมี **Recent invoices** (ใบล่าสุด 3 ใบ) + แผง **Benefit usage · {ปี}** + **Recent activity**
4. สมาชิกที่ยังไม่มีข้อมูล (บัญชียังไม่ผูกกับบริษัทสมาชิก) จะเห็นการ์ดต้อนรับ **"Welcome to the … member portal"** พร้อมปุ่ม **"Explore your benefits"** แทนตัวเลขศูนย์

> ℹ️ วันที่บนหน้าจอ ภาษาไทยแสดงเป็น **พ.ศ.** (display-only) แต่ระบบเก็บเป็น ค.ศ. เสมอ

---

## งานที่ 3 — ดูใบแจ้งหนี้ของบริษัท + ดาวน์โหลด PDF

**สถานการณ์:** อยากดูใบแจ้งหนี้/ใบเสร็จย้อนหลัง

1. เข้าเมนู **Invoices** (`/portal/invoices`) → เห็นตารางใบแจ้งหนี้ของ **บริษัทคุณเท่านั้น** (เลขเอกสาร, สถานะ, วันที่ออก/ครบกำหนด, ยอดรวม)
2. กรอง/ค้นหาได้ด้วยแถบ **Filters** ด้านบน — เลือกสถานะ (Issued / Paid / Overdue / Void / Credited / Partially credited) หรือพิมพ์ค้นหา
   - 🔒 สมาชิก **ไม่เห็นใบร่าง (Draft)** — เป็นของเจ้าหน้าที่
3. กดเลขเอกสารเพื่อเปิด **หน้ารายละเอียด** (อ่านอย่างเดียว) — เห็น line items, subtotal, VAT, total, วันที่ และเลขใบเสร็จ (ถ้าจ่ายแล้ว)
4. ดาวน์โหลด PDF จากปุ่มในคอลัมน์ Actions หรือมุมบนของหน้ารายละเอียด:
   - **"Invoice"** — ใบกำกับภาษี (ยังไม่จ่าย)
   - **"Receipt"** — ใบเสร็จรับเงิน (จ่ายแล้ว แบบแยกเลข)
   - **"Tax invoice / Receipt"** — เอกสารรวม (จ่ายแล้ว แบบรวมเลข)
   - **"Voided invoice"** — ใบที่ถูกยกเลิก (มีตราประทับ VOID)
5. ถ้าใบเสร็จกำลังจัดทำ จะขึ้น **"Receipt preparing…"** — รอสักครู่แล้วลองใหม่
6. อยากให้ส่งสำเนาเข้าอีเมลอีกครั้ง: กดปุ่ม **"Email me a copy"** (ไอคอนซองจดหมาย) — ระบบจะส่งใหม่เข้าอีเมลที่มีในระบบ

> ℹ️ ไฟล์ PDF เป็น source-of-truth — โหลดกี่ครั้งก็ได้เนื้อหาเดิมเป๊ะ
> ⚠️ ใบที่ยกเลิก (Void) จะมีแถบ **"Invoice voided"** + ข้อความ *"This invoice is no longer payable"* — ไม่ต้องทำอะไรเพิ่ม

## งานที่ 4 — จ่ายใบแจ้งหนี้ออนไลน์ (บัตร / PromptPay)

**สถานการณ์:** มีใบสถานะ **Issued** อยากจ่ายผ่านเว็บเลย (ต้องเปิด online payment ที่ฝั่งหอการค้า)

1. เปิดใบสถานะ **Issued** → กดปุ่ม **"Pay now"** (ด้านล่างหน้ารายละเอียด)
   - ถ้าปุ่มไม่ขึ้นแต่เห็นการ์ด **"Online payment unavailable"** = หอการค้ายังไม่เปิดจ่ายออนไลน์สำหรับใบนี้ → ใช้ปุ่ม **"Contact administrator"** เพื่อขอวิธีโอนผ่านธนาคาร
2. ลิ้นชัก (drawer) **"Pay invoice"** เปิดขึ้น แสดง **Order summary** (เลขใบ + ยอด **Amount due**)
3. เลือกวิธีจ่ายจากแท็บ:
   - **Card** — กรอกข้อมูลบัตรในฟอร์มที่ปลอดภัยของ Stripe (ถ้าธนาคารขอ 3D Secure จะมีหน้าต่างยืนยันของธนาคารขึ้นมา ทำตามขั้นตอน)
   - **PromptPay** — สแกน **QR code** ด้วยแอปธนาคารไทย (มีนับเวลาถอยหลังก่อน QR หมดอายุ; ถ้าหมดอายุกด **"Refresh QR"** ออกใหม่)
4. เมื่อจ่ายสำเร็จ จะขึ้น **"Payment received"** + สถานะใบเปลี่ยนเป็น **Paid** ทันที และระบบ **ส่งใบเสร็จเข้าอีเมล** ให้
5. ดาวน์โหลดใบเสร็จได้จากปุ่ม **"Download receipt"** ในหน้าสำเร็จ หรือกลับไปหน้ารายละเอียดแล้วกด **"Receipt"**

> 🔒 **ความปลอดภัย:** ข้อมูลบัตรประมวลผลโดย **Stripe** โดยตรง — เว็บหอการค้าไม่เคยเก็บเลขบัตรของคุณ (มีป้าย **"Secured by Stripe · Encrypted · PCI-DSS compliant"**)
> ⚠️ PromptPay: **สแกนเฉพาะ QR ที่แสดงเท่านั้น อย่าโอนเองไปบัญชีอื่น** (มีคำเตือนนี้ใต้ QR)
> ℹ️ ปิดแท็บกลางคันแล้วกลับมาใหม่ได้ — ระบบใช้รายการเดิม ไม่เกิดการเรียกเก็บซ้ำ
> ℹ️ ถ้าใบถูกยกเลิก (Void) หลังคุณจ่าย ระบบ **คืนเงินอัตโนมัติ** และขึ้นแถบ **"Your payment has been refunded"** พร้อมหมายเลขอ้างอิงการคืนเงิน

## งานที่ 5 — ดูและแก้โปรไฟล์บริษัท + ข้อมูลติดต่อ

**สถานการณ์:** เปลี่ยนเบอร์โทร/เว็บไซต์ หรือเพิ่มเพื่อนร่วมงานเข้าระบบ

### 5.1 ดูโปรไฟล์
1. เข้าเมนู **Profile** (`/portal/profile`) — เห็นการ์ด **Organisation** (ชื่อบริษัท, เลขสมาชิก, Tax ID, ประเทศ, website), **Membership** (plan + ปี, วันลงทะเบียน), **Contacts** (รายชื่อผู้ติดต่อ)

### 5.2 แก้ข้อมูลที่อนุญาต
1. กดปุ่ม **"Edit Profile"** (มุมขวาบน) → หน้า `/portal/edit`
2. แก้ได้เฉพาะช่องเหล่านี้: **First Name**, **Last Name**, **Phone**, **Preferred Language** (ข้อมูลผู้ติดต่อของคุณ) และ **Website**, **Description** (ข้อมูลบริษัท)
3. กด **"Save Changes"** → ขึ้น **"Profile updated successfully."**

> 🔒 ช่องที่แก้ไม่ได้ (plan, tax ID, สถานะ, ประเภทนิติบุคคล, ประเทศ, อีเมล) จะ **ไม่แสดงในฟอร์มแก้ไขเลย** (ไม่ใช่ขึ้นเทาๆ) — เป็นของเจ้าหน้าที่ ถ้าต้องแก้ให้ติดต่อหอการค้า
> ⚠️ **เปลี่ยนอีเมลเข้าระบบไม่ได้** ผ่านพอร์ทัล — ต้องแจ้งเจ้าหน้าที่

### 5.3 เชิญเพื่อนร่วมงาน (เฉพาะ Primary contact)
1. ในการ์ด **Contacts** ของหน้า Profile กดปุ่ม **"Invite Colleague"** (ขึ้นเฉพาะถ้าคุณเป็น **Primary** contact)
2. กรอก **First Name / Last Name / Email / Role / Title / Preferred Language** → กด **"Send Invitation"**
3. ขึ้น **"Invitation sent successfully."** — เพื่อนจะได้อีเมลเชิญ และเมื่อ activate จะถูกผูกกับ **บริษัทเดียวกัน**

> 🔒 ถ้าคุณ **ไม่ใช่ Primary contact** จะไม่เห็นปุ่มนี้ (ขึ้นข้อความ *"Only the primary contact can invite colleagues."* ถ้าเข้า URL ตรง)

## งานที่ 6 — ตั้งค่าบัญชี (รหัสผ่าน · ภาษา · ความเป็นส่วนตัว · ออกจากระบบ)

**สถานการณ์:** เปลี่ยนรหัสผ่าน เปลี่ยนภาษาอีเมล หรือ export ข้อมูลของตัวเอง

เข้าเมนู **Account** (`/portal/account`) — เป็น hub แบ่งเป็น 5 ส่วน:

### 6.1 Account — เปลี่ยนรหัสผ่าน
1. ที่การ์ด **"Account"** กรอก **Current password** + **New password** → กด **"Change password"**
2. ขึ้น **"Password updated. Other sessions have been signed out."** (อุปกรณ์อื่นที่ล็อกอินอยู่จะถูกออกจากระบบทั้งหมด เครื่องนี้ยังอยู่)
3. ลืมรหัสผ่าน: กด **"Forgot your password?"** → ไปหน้า `/forgot-password` กรอกอีเมลรับลิงก์รีเซ็ต

### 6.2 Notification language — ภาษาอีเมลที่ได้รับ
1. ที่การ์ด **"Notification language"** เลือกภาษา (หรือ **"Use chamber default"**) → กด **"Save preference"**
2. ขึ้น **"Notification language updated."** — มีผลกับอีเมลที่ระบบส่งให้ (ใบแจ้งหนี้ รีเซ็ตรหัส ฯลฯ)

### 6.3 Data & privacy — export ข้อมูลของฉัน (GDPR / PDPA) — *เปิดเมื่อหอการค้าเปิด F9*
1. ที่การ์ด **"Data & privacy"** กดปุ่ม **"Request my data export"**
2. ขึ้น **"Export requested"** — ระบบจัดทำไฟล์ในเบื้องหลัง (ปกติไม่กี่นาที) — สถานะจะไล่จาก **Preparing → Ready to download**
3. เมื่อพร้อม กดปุ่ม **"Download"** เพื่อโหลดไฟล์ (โปรไฟล์ ผู้ติดต่อ ใบแจ้งหนี้+PDF events broadcasts และ activity-log ที่เกี่ยวกับคุณ)

> ⏱️ ลิงก์ดาวน์โหลด **หมดอายุใน 1 ชั่วโมง** หลังไฟล์พร้อม (เพื่อความปลอดภัย) — ถ้าหมดอายุให้กดขอ export ใหม่
> 🔒 export ได้เฉพาะ **ข้อมูลของคุณเอง** — ขอข้อมูลสมาชิกอื่นไม่ได้

### 6.4 Renewal reminders + Appearance + ออกจากระบบ
- **Renewal preferences** — เปิด/ปิดอีเมลแจ้งเตือนต่ออายุ
- **Appearance** — สลับธีมสว่าง/มืด (Theme toggle) + ปุ่ม **"Sign out"** ออกจากระบบ

## งานที่ 7 — ดูสิทธิประโยชน์ + ส่ง E-Blast (ถ้ามีในแพ็กเกจ)

**สถานการณ์:** อยากรู้ว่าสิทธิ์ในแพ็กเกจใช้ไปเท่าไร และส่งอีเมลกลุ่ม (E-Blast)

1. เข้าเมนู **Benefits** (`/portal/benefits`) — แท็บ **Benefits** แสดง **Benefit usage · {ปี}** เช่น E-Blast **"2 of 6 used"** + วันที่ใช้ล่าสุด พร้อมลิงก์ลงมือทำ
2. ถ้าใช้ไม่คุ้ม จะมีคำเตือน under-use + ปุ่มพาไปใช้สิทธิ์ (เช่น compose E-Blast)
3. แท็บ **Broadcasts** (ถ้าเปิด F7) แสดงประวัติ E-Blast + โควตา (**Used / Reserved / Remaining / Plan limit**)
4. ส่ง E-Blast ใหม่: ไปหน้า **"Compose E-Blast"** (`/portal/broadcasts/new`) → กรอก **Subject** + **Message** + เลือก **Recipients** → กด **"Submit for review"** (เจ้าหน้าที่อนุมัติก่อนส่งจริง — ยกเลิกได้จนกว่าจะอนุมัติ)

> ℹ️ เมนู Benefits/E-Blasts แสดงเฉพาะเมื่อแพ็กเกจมีสิทธิ์ E-Blast และหอการค้าเปิดฟีเจอร์ไว้

## ❓ คำถามที่พบบ่อย / ข้อควรรู้

| คำถาม | คำตอบ |
|---|---|
| เข้าระบบที่ไหน? | สมาชิกที่ **`/portal/sign-in`** (เจ้าหน้าที่ใช้ `/admin/sign-in` คนละหน้า) |
| ลิงก์เชิญหมดอายุ/ใช้แล้ว ทำไง? | แจ้งเจ้าหน้าที่ออกคำเชิญใหม่ — ลิงก์ใช้ได้ครั้งเดียว |
| ทำไมเห็นแต่บริษัทตัวเอง? | by design — สมาชิกเห็นเฉพาะข้อมูลบริษัทตัวเอง เปิด URL บริษัทอื่นได้ 404 |
| แก้ plan / tax ID / อีเมล เองได้ไหม? | ไม่ได้ — เป็นของเจ้าหน้าที่ ช่องเหล่านี้ไม่แสดงในฟอร์มแก้ไข ติดต่อหอการค้า |
| ไม่เห็นปุ่ม **Pay now**? | หอการค้ายังไม่เปิดจ่ายออนไลน์ของใบนี้ → กด **"Contact administrator"** ขอวิธีโอนธนาคาร |
| ไม่เห็นปุ่ม **Invite Colleague**? | คุณไม่ใช่ Primary contact — เฉพาะ Primary เชิญเพื่อนได้ |
| ไม่เห็นส่วน **Data & privacy**? | หอการค้ายังไม่เปิด F9 (export) หรือบัญชียังไม่ผูกกับบริษัท |
| ลิงก์ดาวน์โหลด export กดไม่ได้? | ลิงก์หมดอายุ 1 ชม. หลังพร้อม — กดขอ export ใหม่ |
| จ่ายเงินแล้วบัตรของฉันถูกเก็บไหม? | ไม่ — ประมวลผลที่ Stripe เว็บหอการค้าไม่เคยรับเลขบัตร (SAQ-A) |
| เปลี่ยนอีเมลเข้าระบบ? | ทำเองไม่ได้ — แจ้งเจ้าหน้าที่ |

## 🔴 สิ่งที่ "ย้อนกลับไม่ได้" / ต้องระวัง

1. **Activate account** — ลิงก์เชิญใช้ได้ครั้งเดียว เมื่อเปิดบัญชีแล้วลิงก์เดิมใช้ซ้ำไม่ได้ (ใช้ `/portal/sign-in` แทน)
2. **Change password** — เปลี่ยนแล้ว **session อื่นทั้งหมดถูกออกจากระบบ** ทันที
3. **Submit for review (E-Blast)** — ส่งให้อนุมัติแล้ว **แก้เนื้อหาไม่ได้** (ยกเลิกได้จนกว่าเจ้าหน้าที่จะอนุมัติ)
4. **ลิงก์ดาวน์โหลด data export** — หมดอายุ 1 ชั่วโมง (ต้องขอใหม่ ไม่ต่ออายุ)
